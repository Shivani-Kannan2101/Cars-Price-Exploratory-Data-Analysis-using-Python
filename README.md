**Car Price Exploratory Data Analysis using Python**

**Project Overview**
This project performs Exploratory Data Analysis (EDA) on a car dataset to understand the factors influencing vehicle prices.

**The analysis includes:**
Data Cleaning
Missing Value Analysis
Duplicate Removal
Statistical Summary
Outlier Detection
Correlation Analysis
Data Visualization

**Python libraries used:**
Pandas
NumPy
Matplotlib
Seaborn

**Objectives:**
Understand the dataset structure
Clean and prepare the data
Explore numerical and categorical features
Identify relationships between engine specifications and vehicle price
Detect outliers
Generate business insights

**Business Questions Answered:**
Which vehicle features have the strongest relationship with car prices?
Does higher horsepower generally lead to higher vehicle prices?
How does fuel efficiency vary across different cars?
Which transmission type is the most common?
Are there any significant outliers in horsepower or price?
Which numerical features are highly correlated?

**Key Findings:**
The dataset contained very few duplicate records.
Engine horsepower shows a positive relationship with vehicle price.
Cars with higher horsepower generally have lower fuel efficiency.
Luxury and performance vehicles appear as outliers in the price distribution.
Highway MPG and City MPG are strongly positively correlated.
