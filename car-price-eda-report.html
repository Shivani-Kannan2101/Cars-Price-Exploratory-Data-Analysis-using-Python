STEP 05Outliers
Detect & Remove Outliers (IQR Method)
Use boxplots and the IQR rule to handle extreme values
PYTHON CODE
# Boxplot for Price (MSRP)
sns.boxplot(x=df['Price'])
plt.show()

# Boxplot for Engine HP
sns.boxplot(x=df['HP'])
plt.show()

# Select numeric columns for IQR check
l = df.dtypes.loc[
    (df.dtypes == "int64") | (df.dtypes == "float64")
].index.tolist()

# IQR calculation
Q1 = df[l].quantile(0.25)
Q3 = df[l].quantile(0.75)
IQR = Q3 - Q1

# Remove rows outside 1.5×IQR range
df2 = df[~((df[l] < (Q1 - 1.5 * IQR)) |
           (df[l] > (Q3 + 1.5 * IQR))).any(axis=1)]

print(df.shape, df2.shape)
OBSERVATIONS & EXPLANATION
▸
Boxplot for Price
MSRP ranges ~₹10,000 to ₹120,000. Median price ~₹65,000, with the middle 50% of cars priced between ₹40,000–₹90,000. No major outliers visible after cleaning.
▸
Boxplot for Engine HP
Horsepower ranges 70–500 HP. Median is around 270–280 HP, with the middle 50% falling between 180–380 HP.
▸
IQR Method
Q1 (25th percentile) and Q3 (75th percentile) define the IQR = Q3 − Q1. Any value beyond Q1 − 1.5×IQR or Q3 + 1.5×IQR is flagged as an outlier and removed into a separate cleaned DataFrame df2.
